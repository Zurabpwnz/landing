TravelChain — es una plataforma blockchain con código abierto manejada por todos los participantes del mercado
El usuario instala la aplicación o widget y decide cuáles son los datos que esta aplicación puede recoger y de qué manera lo hará
Datos del usuario en las aplicaciones
Datos de los servicios turísticos en la web
Datos de las agencias turísticas y agregadores
Los datos pueden guardarse abiertos a todos, despersonalizados o cifrados con la autorización para una entrada y salida
